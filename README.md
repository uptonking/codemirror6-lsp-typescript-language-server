-# codemirror6-lsp-typescript-language-server

> implement codemirror 6 autocomplete/hover/go-to-definition with LSP

## features

- autocomplete
- hover info
- go to definition
- lint

## quickstart

```shell
npm i
npm run start
```

- open http://localhost:8999/
